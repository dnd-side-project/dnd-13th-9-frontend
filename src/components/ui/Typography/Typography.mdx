## 사용법 예시

```tsx
// 기본 Typography 컴포넌트 사용
<Typography variant="title-xl-bold-20">제목</Typography>

// 미리 만들어진 컴포넌트 사용 (권장)
<Title2xl>큰 제목</Title2xl>
<TitleXl>중간 제목</TitleXl>
<BodyL>본문 텍스트</BodyL>
<BodyM>작은 본문</BodyM>

// 태그 변경
<Title2xl as="h1">메인 제목</Title2xl>
<BodyL as="label">라벨 텍스트</BodyL>

// 추가 스타일
<TitleXl className="text-blue-600">파란색 제목</TitleXl>
<BodyM className="text-gray-500">회색 본문</BodyM>
```
